# 스마트 도서 관리 시스템 (Library Management System)

파이썬의 **객체 지향 프로그래밍(OOP)**과 **메모리 최적화**를 실습하기 위한 도서 관리 시뮬레이터입니다. 상속 구조를 활용해 다양한 타입의 도서를 정의하고, 실무적인 메모리 관리 기법인 `__slots__`를 적용했습니다.

### 주요 기능
* **상속 및 다형성**: `Item` 부모 클래스를 기반으로 `Book`(종이책), `EBook`(전자책)을 효율적으로 확장 구현했습니다.
* **메모리 최적화**: `__slots__`를 사용하여 객체 생성 시 발생하는 메모리 오버헤드를 최소화했습니다.
* **타입 검증**: `isinstance`를 활용하여 `Library`에 유효한 객체만 추가되도록 안정성을 높였습니다.
* **캡슐화**: 언더바(`_`)를 활용한 변수 설정으로 데이터 접근 권한을 관리합니다.

### 실행 방법 및 학습 포인트

# 1. 저장소 클론 (내 컴퓨터로 코드 복사)
git clone https://github.com/sujih7-hwang/python-library-management.git

# 2. 프로젝트 폴더로 이동
cd python-library-management

# 3. 프로그램 실행
python main.py
